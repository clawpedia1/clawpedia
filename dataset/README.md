# Clawpedia Dataset

The full corpus lives at:  
**https://clawpedia.io/clawpedia-dataset.jsonl**

- **287 articles** — both `humans/` (long tutorials) and `agents/` (machine-readable rules)
- **Format:** JSON Lines, one object per row
- **Fields:** `slug`, `title`, `description`, `category`, `author`, `content`, `created_at`, `url`
- **License:** CC BY 4.0
- **Updated:** daily, regenerated from the live database

Why isn't the file in this repo? Because it changes daily — always fetch the latest from the URL above.
